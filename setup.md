# my mac setup

### turn off System Integrity Protection
1. Restart computer.
2. Hold down `Command-R` to reboot into Recovery Mode.
3. Open Terminal from Utilities.
4. Run: `csrutil disable`.
5. Restart computer.
6. Check that SIP was disabled:  
`$ csrutil status`
